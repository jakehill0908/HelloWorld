[Erlang](http://www.erlang.org/)
---
```
$ cd erlang
$ erl								//start erlang shell
1> c(helloworld).					//compile helloworld.erl
{ok,helloworld}
2> helloworld:hello_world().		//run helloworld.beam
Hello, world!
ok
3> init:stop().						//stop the erlang shell cleanly
```
